I build full-stack applications with a focus on clean architecture and maintainability — robust backend systems in **.NET** and type-safe frontend apps with **Next.js**.
 
On the backend, I design modular systems using **CQRS**, **MediatR**, and layered architecture, with secure APIs backed by **JWT + RBAC**, **EF Core** data access, and observability via **OpenTelemetry**. On the frontend, I build server-component-first applications with **TanStack Query**, **Zod**, and **shadcn/ui**.
